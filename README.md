## :toolbox: Getting Started

### :gear: Setup

**Clone the repository**

```bash
$ git clone https://github.com/AlxisHenry/url.alexishenry.eu.git
```

### :gear: Configuration

```bash
$ composer install
$ npm install
$ npm run build
$ php artisan migrate:fresh --seed
$ php artisan serve
```

### :test_tube: Tests

**Run the tests using the following command**

```bash
php artisan test
```
